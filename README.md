# 📋 Sistema de Control de Asistencias

Sistema web desarrollado para gestionar y controlar la asistencia de estudiantes mediante diferentes roles de usuario, permitiendo la administración de docentes, estudiantes, materias, grados y registros de asistencia.

## 🚀 Características

- Gestión de usuarios.
- Gestión de estudiantes.
- Gestión de docentes.
- Gestión de grados.
- Gestión de materias.
- Registro y consulta de asistencias.
- Generación de códigos QR.
- Panel de administración.
- Acceso según roles de usuario.

## 🛠️ Tecnologías Utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap
- JavaScript
- QR Code Generator

## 👥 Roles del Sistema

### Administrador

- Gestionar usuarios.
- Gestionar docentes.
- Gestionar estudiantes.
- Gestionar materias.
- Gestionar grados.
- Consultar asistencias.

### Docente

- Registrar asistencia.
- Consultar estudiantes asignados.

### Estudiante

- Consultar información personal.
- Consultar registros de asistencia.

## 🎯 Objetivo del Proyecto

Facilitar el control y seguimiento de la asistencia académica mediante una plataforma web centralizada, mejorando la gestión administrativa y el acceso a la información.

## 📈 Mejoras Futuras

- Estadísticas de asistencia.

- Integración con correo electrónico.
